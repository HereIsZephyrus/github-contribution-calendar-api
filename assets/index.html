GitHub Contribution Calendar API

https://githubapi.ryanchristian.dev/user/{username}

Data Structure

Below is an example of the data structure. `total` is a count, `contributions` is an array of arrays made up of 52-53 arrays, each holding 7 values representing a date. This API will cache values, meaning counts are only updated once per 24 hours per user.

{
  total: 2328,
  contributions: [
    [
      {
        date: "2019-09-01",
        count: 8,
        intensity: 1
      },
      {
        date: "2019-09-02",
        count: 2,
        intensity: 1
      },
      {
        date: "2019-09-03",
        count: 7,
        intensity: 1
      },
      ...
    ],
    ...
  ]
}
